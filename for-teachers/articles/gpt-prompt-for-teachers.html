20 Best ChatGPT Prompts for Teachers

These prompts are designed to help teachers prepare high-quality lesson plans, worksheets, assessments, classroom activities, and learning resources. Simply copy a prompt, replace the text inside square brackets [ ] with your own information, and paste it into ChatGPT.

Prompt 1 – Create a Complete Lesson Plan

Purpose Create a detailed lesson plan for any subject.
Prompt You are an experienced teacher with more than 20 years of teaching experience. Create a detailed lesson plan for [Subject], Class [Class], Chapter [Chapter Name]. Include learning objectives, prior knowledge, teaching-learning materials, introduction, explanation, classroom activities, real-life examples, higher-order thinking questions, assessment questions, homework, learning outcomes, and teaching tips. Write in simple language suitable for students.
Example Class 8 Science – Crop Production and Management

Prompt 2 – Generate a Complete Question Paper

Purpose Create school examination papers.
Prompt Prepare a complete question paper for Class [Class] [Subject] based on the latest CBSE pattern. Include MCQs, very short answer questions, short answer questions, long answer questions, case-study questions, competency-based questions, assertion-reason questions, internal choice, answer key, and marking scheme. Maintain proper difficulty level and board exam standards.
Example Class 10 Mathematics

Prompt 3 – Create a Student Worksheet

Purpose Create printable worksheets.
Prompt Create a classroom worksheet for Class [Class], [Subject], Chapter [Chapter Name]. Include fill in the blanks, true/false, match the following, multiple-choice questions, short answer questions, long answer questions, higher-order thinking questions, and one fun classroom activity. Also provide the answer key.
Example Class 6 Mathematics – Fractions

Prompt 4 – Explain Any Topic Like an Expert Teacher

Purpose Understand difficult concepts easily.
Prompt Explain the topic [Topic Name] for Class [Class] students in very simple language. Teach like an experienced classroom teacher. Use daily life examples, simple diagrams wherever required, important definitions, key points, memory tricks, common mistakes made by students, and five practice questions with answers.
Example Photosynthesis – Class 7 Science

Prompt 5 – Create a Complete Chapter Revision Package

Purpose Prepare students before examinations.
Prompt Prepare a complete revision package for Class [Class], [Subject], Chapter [Chapter Name]. Include chapter summary, important definitions, formulas (if applicable), important dates, important diagrams, one-mark questions, two-mark questions, three-mark questions, five-mark questions, competency-based questions, important MCQs, previous year question types, common mistakes, quick revision notes, and 20 practice questions with answers.
Example Class 9 Social Science – The French Revolution

Tips for Better Results

  • Always mention the class, subject, and chapter name.
  • Specify the education board (CBSE, ICSE, State Board, etc.).
  • Ask for simple language if the content is for younger students.
  • Request step-by-step explanations for mathematical and scientific topics.
  • Ask ChatGPT to follow the latest syllabus and examination pattern.
