Q1. Which of the following is a quadratic equation?

(a) x + 5 = 0
(b) x² + 3x + 2 = 0
(c) x³ + 2x = 0
(d) 2/x + 1 = 0

Answer: (b) x² + 3x + 2 = 0

Solution: The highest power of x is 2.

Q2. The roots of x² − 5x + 6 = 0 are

(a) 1, 6
(b) 2, 3
(c) -2, -3
(d) 5, 6

Answer: (b) 2, 3

Solution: x² − 5x + 6 = (x − 2)(x − 3).

Q3. The standard form of a quadratic equation is

(a) ax + b = 0
(b) ax² + bx + c = 0
(c) ax³ + bx² + c = 0
(d) ax² + b = 0

Answer: (b) ax² + bx + c = 0

Q4. The value of the discriminant of x² − 4x + 4 = 0 is

(a) 16
(b) 0
(c) 4
(d) -4

Answer: (b) 0

Solution: D = b² − 4ac = (-4)² − 4(1)(4) = 0.

Q5. If D > 0, the quadratic equation has

(a) No real roots
(b) Equal roots
(c) Two distinct real roots
(d) One root

Answer: (c) Two distinct real roots

Q6. If D = 0, the roots are

(a) Imaginary
(b) Equal
(c) Distinct
(d) Irrational

Answer: (b) Equal

Q7. If D < 0, the quadratic equation has

(a) Equal roots
(b) Distinct real roots
(c) No real roots
(d) One real root

Answer: (c) No real roots

Q8. The roots of x² − 9 = 0 are

(a) ±9

(d) -3 only

Answer: (b) ±3

Solution: x² = 9 ⇒ x = ±3.

Q9. The sum of roots of x² − 7x + 10 = 0 is
